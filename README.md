# Vegi-Plan

Smart vegetarian nutrition tracker and meal planner.

## Tech Stack

- Next.js 16 + React 19 + TypeScript
- Tailwind CSS v4 + shadcn/ui (new-york)
- Recharts + Framer Motion + lucide-react
- localStorage (no backend)

## Features

- Dashboard with nutrition overview and critical nutrient tracking
- Weekly meal planner with recipe assignment
- Daily meal logging with per-meal nutrition summary
- 18 vegetarian recipes with bilingual support (EN/KO)
- Ingredient scanner (mock OCR analysis)
- Dark mode + language toggle (English/Korean)
- Settings with calorie targets and vegetarian type selection

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open http://localhost:3000
